# Platform skill template

Copy this when adding a new OTA platform to the opencli toolchain (e.g. Viator, Airbnb Experiences, Expedia). **Do not edit this file in place** — copy the content into `.claude/skills/opencli-<platform>/SKILL.md` and fill in the TODOs.

For an existing platform with a scaffold stub already present (trip / getyourguide / kkday), edit that stub directly instead of using this template. For a fully filled-in example, see `.claude/skills/opencli-klook/SKILL.md`.

---

## How to use

1. Decide the short platform slug (lowercase, matches `src/clis/<slug>/`). Example: `viator`.
2. Create the skill directory via `skill-creator`:
   ```bash
   python3 /Users/ryan.huang/.claude/plugins/cache/anthropic-agent-skills/document-skills/unknown/skills/skill-creator/scripts/init_skill.py \
     opencli-<slug> --path .claude/skills
   ```
3. Delete the auto-generated `scripts/`, `references/`, `assets/` directories unless you actually need them.
4. Overwrite the generated `SKILL.md` with the template below, then fill in each TODO.
5. Validate:
   ```bash
   python3 /Users/ryan.huang/.claude/plugins/cache/anthropic-agent-skills/document-skills/unknown/skills/skill-creator/scripts/quick_validate.py .claude/skills/opencli-<slug>
   ```
6. Add a row to the Skill ownership table in `CLAUDE.md` and to the Platform dispatch table in `opencli-router/SKILL.md`.
