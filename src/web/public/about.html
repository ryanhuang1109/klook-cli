How It Works 運作原理

Three steps from search to insight 三步驟，從搜尋到洞察

01
🔌
Browser Bridge

Powered by opencli, this tool uses a Chrome extension to borrow your existing login sessions. No credentials stored — if your Chrome is logged into Klook, the tool can access it just like you would.

Browser Bridge 技術

基於 opencli 框架，透過 Chrome 擴充程式借用你的瀏覽器登入 session。不需要提供帳號密碼 — 只要 Chrome 有登入該網站，工具就能像你一樣操作。

02
🔍
Search 4 Platforms

Simultaneously search Klook, Trip.com, GetYourGuide, and KKday. Each platform's activities, prices, ratings, and reviews are extracted in real-time.

搜尋四大平台

同時搜尋 Klook、Trip.com、GetYourGuide、KKday。即時抓取各平台的活動、價格、評分、評論數。

03
🤖
AI Clustering

Results from all platforms are sent to an AI (Claude via OpenRouter) that groups similar products, converts currencies to USD, and identifies the best price and highest rated option per group.

AI 智能分組

將所有平台的結果送給 AI（Claude via OpenRouter），自動將相似產品分組、統一幣別為 USD、標出每組最低價和最高評分。

Features 功能

Everything you need for price intelligence 完整的價格情報工具

📊
POI-Based Monitoring

Configure Points of Interest (like "Mt Fuji day tour" or "USJ tickets") with custom keywords per platform.

POI 監控

設定要監控的景點（如「富士山一日遊」、「USJ 門票」），每個平台可設定不同搜尋關鍵字。

💰
Price Comparison

AI groups similar products across platforms, normalizes to USD, and highlights the cheapest and best-rated options.

跨平台比價

AI 自動將相似產品分組、統一幣別、標出最低價和最高評分。

📈
Historical Tracking

Save comparisons to a local database. Track price changes over days and weeks. Detect promotions and price hikes.

歷史追蹤

比價結果存入本地資料庫，追蹤每日價格變動，偵測促銷和漲價。

🌐
Web Dashboard

BD colleagues can use the web interface — no terminal needed. Manage POIs, run comparisons, view results and history.

Web 介面

BD 同事不需要 terminal，直接用瀏覽器操作 — 管理 POI、跑比價、看結果。

CLI Examples CLI 指令範例

Run from the terminal 在終端機執行

bash 
# Search
opencli klook search "Mt Fuji day tour" --limit 5

# Activity detail with itinerary & packages
opencli klook detail 93901

# Trip.com date comparison
opencli trip detail 92795279 --compare-dates

# AI cross-platform comparison
node dist/cli.js compare "Mt Fuji day tour" --date 2026-04-15

# Price history
node dist/cli.js compare-history "Mt Fuji day tour" --days 7

# Web dashboard
npm run web
# → http://localhost:17890

Setup Guide 安裝指南

Get started in minutes 幾分鐘內開始使用

For BD Colleagues
BD 同事

Just open the shared URL in your browser. No installation needed.

直接用瀏覽器打開分享的 URL 即可，不需要安裝任何東西。

For Engineers
工程師安裝步驟
  1. 1 Install opencli
    npm install -g @jackwener/opencli
  2. 2 Install Browser Bridge extension
    Chrome → Settings → Extensions → Developer Mode → Load unpacked → select the extension folder.
  3. 3 Verify setup
    opencli doctor
  4. 4 Clone & build
    git clone <repo> && npm install && npm run build
  5. 5 Register plugins
    ln -sf <plugin-path> ~/.opencli/plugins/
  6. 6 Set OpenRouter API key
    Add OPENROUTER_API_KEY=... to your .env file.
  7. 7 Log into platforms in Chrome
    Sign in to Klook, Trip.com, GetYourGuide, and KKday in your Chrome browser.
  8. 8 Start using
    opencli klook search "Mt Fuji" or npm run web
